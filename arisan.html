🔥 Setup Firebase
Tempelkan Firebase Config milikmu agar data sinkron ke semua HP.
📋 Cara Dapatkan Config:
  1. Buka console.firebase.google.com
  2. Buat project baru / pilih project yang ada
  3. Klik ⚙️ Settings → Project Settings
  4. Scroll ke bawah → Your apps → tambah Web App (</>)
  5. Copy firebaseConfig yang muncul
  6. Di Firestore → Create Database → test mode
